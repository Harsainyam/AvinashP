üöÄ Quick Start Guide

  1. Start the backend server:
    cd backend && npm run dev
  2. Ensure all services are running:
    • PostgreSQL on port 5432
    • MongoDB on port 27017
    • Redis on port 6379
  3. Run Authentication Tests first to create a user and get a token
  4. Copy the access token from the auth test results
  5. Use the token in Account and Transaction tests
  6. Review test results in each browser tab
